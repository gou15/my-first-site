Interactivity with JavaScript

For ease of coding, and to make sure I get this final project done on time, there will be minimal Bootstrap styling going forward. While Bootstrap does make styling easy, styling is not my forte. We'll continue importing the Bootstrap library but it won't be applied with any regularity until the end.

We learned a lot about JavaScript and, rather than explain it all here. I'm just going to go ahead and demonstrate it then explain what's happening at the end. Here's how this works playa:

  1. Click the button
  2. Do what gets asked of you
  3. Then click on the blue box
  4. The blue box will turn green and tell you a little something
  5. Another box will turn blue - click that one next
  6. Carry on until all boxes are green
  7. Click the link under the 6 boxes to try again

OK so what just happened here? We've got the large button styled by Bootstrap listening for a click. Nothing else works until the button is clicked. Once it's clicked you get a prompt for a positive non-zero integer with some JavaScript validating your input. I couldn't break it, but if you can please let me know. Once it's got a valid entry the rest of the divs (again, styled by Bootstrap) are ready for their events (also a click, but it could be mouseover or something else). We're replacing classes as we click on valid divs to change the background and border colors. We're also writing in not just text but HTML as you're moving along and watching the JavaScript that I wrote play with your number. Retrieving the buttons and divs to have them listen for JavaScript events is based on the DOM which I'm not going to try to explain. If I could explain it well I'd be teaching, not learning.

We start by simply displaying your number. Then we apply a few basic operators to it. Then we get another number from you (with validation) and mash them together in a few different ways. Then we run a little loop based on your number to create the next 1000 numbers in sequence. I displayed this in the console instead of on the document itself to save your eyes. Next I converted your number to a string so I could find out how long it was, tell you the last digits, and insert it as text into a link that I generated (I'm actually super proud of this one). Last I started taking away from your number every second. This decrementing number doesn't stop unless you stop the timer, which you do so by canceling the "Go Again" prompt. There's a lot going on here and it's a ton to try to explain. I added my JavaScript as comments in this HTML file so go ahead and have a look at it. It's not very efficient (I could have used arrays for the div selectors, maybe written a few more functions so I wouldn't have to repeat my code so much, and I have my variables in global scope because I couldn't ge them to work otherwise... next time).
