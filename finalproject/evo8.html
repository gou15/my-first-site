Getting Data Quick and Easy


We're going to pull some data from another location and show it to you.





What's going on here? I'm taking your TV show input and turning into a string variable (removing any spaces and replacing them with plus signs). That variable is being inserted into a URL search string which talks to an API and will return a JSON object containing all sorts of information based on what it finds. I am taking some information from that JSON object and displaying it to you in a user friendly format. If you want to see the full JSON object check your console. You can draw API data from anywhere that's offering it, like Google Maps, and as long as you can interpret the JSON it is easy to present it to your user!
