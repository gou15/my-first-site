Getting Data Quick and Easy


It's been a ride. We went from knowing nothing and having no tools to being equipped to write code, learning how a browser works and how it interprets a markup language, what that interpretation does to the content we want to present, embedding images, audio, video, and other content into our own pages, get information from a user with forms, add some color and style, utilize other people's code to organize and control how our content is presented, and then use a coding language to manipulate not only the data we receive but the markup language itself to dynamically alter our page. We even learned how to pull information in JSON format from an API, interpret that data, and again present it in a user-friendly way.


We've been writing every line of every page ourself up to this point. Surely there's got to be a better way...
